The original

Back after another two week hiatus (sorry about this, traveling across Europe, giving talks, and preparing for a launch while still being a husband and dad can be a bit demanding).

News & announcements

PHP 5.5.0RC1 released
This RC is a fix release for various bugs and memory leak, and in fact all releases up until 5.5 goes gold are as it's in feature freeze.

PHP 5.4.15 and PHP 5.3.25 really were released
There was some confusion when 5.4.15 and 5.3.24 were announced as a glitch in propagating the releases to the mirrors  stopped them from being available, but it's all fixed now so get 'em while they're hot.

Google made PHP available as a runtime on AppEngine
As others have said this week Google have finally made the language that powers 75% of the web available on their popular platform as a service, AppEngine, which previously was only capable of running Python and Java apps. See the Google I/O blog post for more details. Along with the announcement from the Goog, JetBrains announced the immediate release of a plugin which adds AppEngine support to the popular PHPStorm IDE, with built-in support to follow quickly.

Guzzle 3.5.0 released
One of the best HTTP/HTTPS client libraries for PHP around has got a major version bump, with memory and speed improvements, as well as scores of fixes and some refactoring. See the release notes for more.

Yii 2 Public Preview Available

Audiovisuals

PHP Townhall episode 7
Regular hosts Phil Sturgeon ♥ and Ben Edmunds ♥ are joined by Ratcher creator and React co-creator Chris Boden ♥, and guess what, wouldn't you know they spend most of the time talking about WebSockets and non-blocking async/IO in PHP? Go figure. This is a fascinating episode as async/non-blocking APIs are one of my main areas of interest and I've wanted to know more about React and Ratchett for a while, and where better than the horses mouth, so to speak? The guys also talk about a few other bits and pieces, including the first ever PHP Townhall meetup, which while being held online didn't stop everyone from getting drunk and ranting for several hours. Good work.

Lately in PHP podcast episode 35
Regular hosts Manuel Lemos ♥ and Ernani Joppert ♥ talk about the need for better PHP internals docs, especially with the merge of the new shiny opcode cache, faster opcodes, interested rejected RFCs, and Manuel's recent article on version control systems.

Behat & Beautiful APIs talk slides
Slides from two talks by Tobias Schlitt, first on using Behat for behavioural driven development (held as a "webinar"), and the second a talk on API design given at the Cologne Symfony Usergroup.

Development, by the Numbers talk ♥
Slides and video from Anthony Ferrara's talk at DrupalCon Portland on development best practices and tooling. Great talk by a great guy, go and watch it.

Articles

PHP on Google App Engine – Quick First Review
A thorough write up on the new PHP Runtime on AppEngine, from what popular extensions are and aren't available, to some simple examples of getting an app up and running. If you've not had time to go over any of the docs or press releases from the Goog then this article makes for a pretty good primer.

Google App Engine, PHP and Silex. Setting up a Login Provider ♥
We're going to see lots more real world examples and reusable libraries just for GAE over the coming weeks, case in point Gonzalo Ayuso not only guides us through allowing logins through the GAE auth system, but also provides this full reusable Silex auth provider for it.

7php - A Chat About Adminer ♥
The prolific Khayrattee Wasseem did this follow up to his earlier interview with Jakub Vrana, this time discussing Jakub's database administration tool Adminer, a small and efficient alternative to the infamous phpMyAdmin.
Although I'm a command line guy myself so never really have use for these sorts of tools, but I know a lot of people out there prefer a more visual interface, or are restricted to how you can access a server, so this may be in handy, especially if like Jakub you're tired of copying the large phpMyAdmin codebase around.

7php - Know Thy PHP User Group Know The Leeds PHP User Group (LeedsPHP) ♥
Khayrattee also started this fantastic new feature on 7php taking a closer look at regional PHP user groups, interestingly starting with my geographically nearest group, LeedsPHP, which sadly I must admit despite being close and even having worked for a few years in Leeds I've never managed to make a meeting. However, this post, exploring the vibrancy of the strong user group and how they're managing to keep each meeting open and welcoming to everyone, has encouraged me to attend the next one!

Introduction to Vagrant/Puppet and introducing PuPHPet ♥
Vagrant is possibly one of my favourite tools of the last few years, and it is perfectly matched with configuration management tools like Puppet and Chef, however as Juan Treminio points out in this introductory post while vagrant itself is amazingly easy to use many people who start out trying to use Puppet to configure PHP (and the rest of the LAMP stack) in a VM can struggle, so he's created a great online GUI tool called PuPHPet to configure and generate a Vagrantfile and Puppet manifest to get you going. The code for PuPHet is also open and available on GitHub.

Cloning in PHP ♥
Giorgio Sironi explores the various ways in which a value can be a copied, including object instances which are cloned, and how cloning can be programmatically overridden for a class. Good overview if you're familiar with the different nuances beyond references and copy-on-write.

First PHP meetup by PHP Srbjia in Belgrade ♥
Miro Svrtan writes about the first ever PHP Srbjia PHP meetup (although with 250+ attendees and 4 talks it was more like a full conference), held in Belgrade on 28th of April. It looks like the day was an overwhelming success, and as Miro says it seems to be something the PHP community over there really needs, and it's very good to see community organised events like these take off.

Composer and PSR-0: Friends, Not Relatives ♥
Phil Sturgeon clears up a lot of confusion people have over the relationship between Composer and PSR-0, covering in detail (and his usual sarcastic wit) each common misconception and the technical reasons for why something might be (if indeed it is that way).

Static code analysis tools for PHP ♥
This well written article by Christopher Louis Martinez covers the wealth of static code analysis tools for PHP that allow the analysis and detection of various code smells that could cause issues down the line, even if it's just maintainability from badly formatted code.
Christopher also makes a good argument for the case that "good code" should be statically analysable, and that an inability to do this can be a code smell by itself.

Improve PHP session cookie security ♥
A quick but useful post by Simon Holywell covering a few of the INI/runtime options PHP provides to help armour a session from various attacks.

Compiling PHP 5.5 From Scratch ♥
A great write up by Brandon Savage detailing configuring and compiling 5.5, while we wait for the release to go gold and make it's way into various package managers; although his instructions are for a Debian based Linux system (such as Ubuntu) it shouldn't be too difficult to follow the same steps on another other nix-like.

#phpuceu ♥ ♥
Ole Michaelis and Soenke Ruempler attended the 2nd PHP Unconference in Europe, and had a fantastic time by the looks of it. They detail some of the adhoc talks they went on throughout the weekend, and great discussions. I love the fact that most of the unconference took place outside in the early summer weather (which has been sadly lacking this side of the atlantic, even in Berlin where the event took place).

Zend Framework 2 – Hydrators, Models and the TableGateway Pattern ♥
Matthew Setter shows us how to implement the hydrator, date mapper and table gateway patterns with Zend Framework 2, prsenting a very nice solution for decoupling your models from the datasource, right down to how fields are mapped (if required). There is some discussion in the comments that is worth checking out as this seems a hot topic in ZF2 at the moment, and the example Matthew provides is a great starting point to wet your lips on.

New in Symfony 2.3: Small things matter 
Symfony grand pubah Fabien Potencier goes over some of the many many small but exciting changes in Symfony 2.3.

Functional Testing to Improve Quality Assurance (part 1) ♥
A good high level view by Andy Podner of what functional testing is, as opposed to unit testing, complete with handy racecar analogy, and a brreakdown of benefits to doing functional testing alongside unit testing.

Code complexity and clean code ♥
Testing and clean code maestro Brandon Savage wrote up this excellent description, with practical examples, of cyclomatic complexity, and NPath complexity, two formal definitions and metrics for measuring code complexity. The two methodologies go hand in hand when doing automated code complexity analysis and a good understanding of them should place most developers in a good position to at the very least rethink the complexity and clealiness of their code.

Simplest PHP Generator Example ♥
Lorna Jane Mitchell really likes generators, which is great as everyone should, generators are awesome and she goes over a simple example (in fact the simplest example) of a generator as you will be able to use them in PHP 5.5, which she backs up with a description of what makes them great and the practical cases when you'd want to use one (pretty much any time you want to use an Iterator, but with less code complexity).

Survived php tek 2013 ♥
A tantalising write up of php|tek 2013 by this year's uncon organiser Michelangelo van Dam, including some of his favourite talks and highlights from other events such as the Hacktathon (kudos to Lorna Jane Mitchell and joind for lots of pull requests).

Understanding HTTP Digest Access Authentication 
Sean Hudgston goes step by step through using HTTP Digest Authentication in PHP, which in my experience can be a bit of a pain in the posterior so good to see more how to's on this for those few instances when you need to handle Digest auth at the application level.

A Conversation About Testing in PHP ♥ ♥
Chris Hertjes and Ed Finkler of /dev/hell fame shoot the breeze about PHP testing tools, methodologies and just why Chris weeps for the future.If you're struggling with any testing concepts, or like Ed don't like PHPUnit very much, read through this and you just made glean some insight. At the very least it's an amusing read. ;-)

Full Test Coverage is Impractical ♥
In this quick post Anna Filina restates a view I've held for many years: "full" test coverage of an application not only means very little in many cases, but is actually impossible (for some values of impossible, i.e. very bloody hard). Not only this the tests created tend to mean very little and provide a false feeling of protection.


Projects

Lurker ♥
A promising library for Symfony2 resource objects that allows you to subscribe to a watcher and receive events when a watched resource changes (or in most cases a certain method is called).

Mondrian ♥
An extremely cool tool for performing static code analysis of a project, along with methods for applying graph theory to code, and generate pretty visualisations of different metrics.

RMT ♥ ♥
A tool for managing relases in a project, allowing for the use of templates to update changelogs, VCS tags and more.

doc-renderer ♥
A pure PHP library for rendering Markdown and RST as HTML.

fabricius ♥
A Jekyll like static site generator with a nice API and the usual goodies (markdown support, content querying etc.)

boris ♥
A small REPL for PHP, written in PHP itself, with some nice features. Very interested in this as I've used Facebook's phpsh REPL for some time, after greatly missing the REPLs from Python and Ruby when working with PHP.

Nyan Cat PHPUnit Result Printer ♥
Ever wanted a Nyancat to dance across your terminal displaying your unit test results? Of course you have, why wouldn't you? Well here you go.
