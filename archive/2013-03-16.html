News & announcements

PHP 5.5.0 alpha 6 released
A mostly bug fix release prior to the Zend Optimizer+ merge and the now longer road to 5.5 going gold. See the NEWS file for more details.

Community voted to delay 5.5 to integrate Zend Optimizer+
Great news for speeding up PHP out of the box (eventually), despite issues with it delaying the release schedule...and guess what? It's now been merged to master. To test the ZO+ opcache with PHP 5.2+ you can use the PECL extension today.

More PHP extensions available on Travis CI
The awesome continuous integration system that more and more open source projects are using to keep their code clean and their tests green (and fair few closed), now has more PHP extensions out of the box, and an easier process for enabling them in the php.ini used by the Travis worker VMs.

Zend Framework 2.1.4, 2.0.8, and 1.12.3 released
These new ZF2 and ZF1 releases fix a bunch of security problems, as well as various API fixes.

The Grumpy Programmer's PHPUnit Cookbook by Chris Hartjes released
Chris Hartjes, aka @grmpyprogrammer, has released a brilliant new ebook all about the sticky subject of unit testing and using good ol' PHPUnit to get the heavy work done. Highly recommended.

PHP-FIG vote to concentrate more on "hard" specs
Expect more so-called "hard" specs, e.g. defining actual interfaces for use in code like the recent Logging interfaces, and less coding standards suggestions.

 

First speakers announced for Symfony Live Portland 2013

Keynote speakers announced for php|tek13

Audiovisuals

Symfony Live videos
Some great videos of past Symfony Live talks have posted to the Symfony YouTube channel, most recently last years Berlin conference.

PHP Townhall regular hosts Phil Sturgeon and Ben Edmunds are joined by Miko Federmann, and /dev/hell regular hosts Chris Hartjes and Ed Finkler. They covered the sticky subjects of the PHPness gate, industry sexism, and Chris's struggles with and tips for dealing with his depression and anxiety, which really struck a personal chord with me.

Regular hosts Manuel Lemos and Ernani Joppert interviewed The PHP Programming Innovation Award Winner of 2012 Karl Holz , and went on to talk about various topics such as the Optimizer+ merge and Manuel's recent blog post comparing PHP to Hobbits.

/dev/hell Episode 29

Articles

PCI Compliance and the PHP Developer
As much as I dislike the "PCI standards" for various reasons, if you're dealing with customer credit card data you need to be aware of them, and this article by  David Shirey does an excellent job of simply explaining the ins and outs of achieving compliance from the standpoint of a PHP developer.

Extending an Iterator to use an Iterator to make your code a little cleaner
Bob Majdak Jr wrote this nice little example of extending Iterators objects in order to wrap the common new ThingIterator(new OtherThingIterator)) pattern you've probably used lots of times in your own code, making it much cleaner and simpler by doing the extra iterator instantiation in the extended class. Good stuff.

Integrate PHP application with Solr search engine
Lukasz Kujawa has written this concise step by step guide to getting Apache Solr up and running and how to use the Solr extension to perform search functions from your PHP application. Solr is a very nice search engine backend and if you need more than a *SQL full text search, like many serious apps do, this is a good route to take.

Scaling Silex applications (part II). Using RouteCollection
Following up his previous post on scaling Silex, Gonzalo Ayuso this time covering the use of the RouteCollection to wrap the underlying Symfony2 routing, not only making your code a bit cleaner but also making it easier to extend your routing without degrading maintainability.

Your One-Stop Guide to Laravel Commands
A fairly exhaustive guide by Jason Lewis to building command interfaces in Laravel 3 with Artisan, including accepting commands, arguments, options and asking for user input.

Building ePub with PHP and Markdown
A very interesting article by Vito Tardia  on generating ePub documents (most commonly used for eBooks) from markdown documents using PHP, various libraries and some very clever tricks and workarounds.

Composer: an update on require-dev
Jordi Boggiano of Composer fame informs us about an update to how Composer handles the "require-dev" field in packages, in that the entire feature has been rewritten by him, should be faster, and much more reliable. We actually noticed this at Server Density in our application builds before I saw Jordi's blog post, and has made our builds a lot simpler between different environments.

Valitron: The Simple Validation Library That Doesn't Suck
Vance Lucas announces his new validation library Valitron and explains his motivations for creating yet another validation library and why you should use it too. Interesting project and definitely one to use if like Vance you're building an app that doesn't already have the same dependencies a lot of other libraries have, or their interfaces frustrate you.

The Wheel: Symfony Console
Seems everyone's coving command libraries this week. In his new feature The Wheel, Giorgio Sironi wrote up the pros, cons, and general usage of the Symfony Console library.

Practical Aspects of the Adapter Pattern
Rakhitha Nimesh succinctly covers the practical uses of the Adapter pattern, with an analogy centring around controlling APIs via text messages, and a more real world PHP example for sending emails and tweets via a compatible interface.

How to Tell if Your PHP Site has been Hacked or Compromised
Steps to Take When you Know your PHP Site has been Hacked
In these two articles by Greg Freeman he goes step by step through various warning signs for a compromised PHP site, and how to respond to the situation. To these tips I would also add to make sure to keep up with application updates if running with an "off the shelf" solution like Wordpress/Drupal etc., and to regularly check updates to your dependencies if it's a bespoke application. If you're using Composer the security checker by SensioLabs is great for this.

Overloading: Create PHP Class Methods on the Fly
Although not strictly "overloading" in the true OO sense of the word, this article by Andy Podner on creating methods and static methods on-the-fly using a registry of anonymous functions with the correct arguments passed from call-time magic methods is a great little primer on a technique that can often make your code simpler and easier to extend.

PHP is the right tool for the job (for all the wrong reasons)
A pragmatic look by Samuel Levy on many of PHP "strengths", which detractors usually either forget about or conveniently concrete over in their arguments.

Unauthorized Access: Bypassing PHP strcmp()
A write up by Daniel Regalado on his attempt to work through the Codegate 2013 security challenge, in which he came across a common vulnerability caused using the strcmp() function to compare strings without checking the type of the value returned, similar to issue with strpos that almost everyone comes across. I commented on the blog why this happens, but I'd just like to add that for the purposes of checking hashes as in the challenge you should just use an equality operator directly rather than something that uses lengths in it's comparison, like strcmp.

Projects

caq
A possibly useful bash script that generates base applications for various frameworks from template files, seems to be easily extendable by defining your own templates and may be just what you need if you're always bootstrapping apps with common base files.

php-o
An API which wraps common scalar values and objects in PHP providing a "saner API", for example: echo s("abc")->len();

e2ex
A package that captures and convert PHP errors/warnings and converts them into actual exceptions, with optional logging via the excellent monolog library.

PHP Cron Expression Parser
A library for generating run-times based on datetime and repeatable configurations, e.g. like the cron unix daemon, with an inbuilt crontab syntax parser.

Box
A library for simplifying the creation or Phar archives, with a very nice API, could be just what you need for generating those phar's in your build steps.

Brainfart
A brilliantly implemented Brainfuck interpreter, self described by the author as being over engineered as he used it as an example for teaching a friend some of the more advanced features in 5.4. Even includes a complete unit test suite. *claps*

Inflector
A port of a popular Ruby library that allows you to perform inflections on English words, mostly with regards to code symbols, e.g. turn CamelCase to snake_case, or turn singular words to plural (great for data models), etc.

XHP
An interesting PHP extension by Facebook which adds XML elements as an actual first class type in PHP. Hard to explain so I'll give an example:
Instead of writing echo '<a href="' . $link . '">' . $title . '</a>';
You might use: echo <a href="{$link}">{$title}</a>;
Instead, and because these are actual PHP expressions inside real XML elements, you can do a lot more complex things beyond string interpolation.

Namespacer
A nice little library for wrapping old underscore separate libraries and accessing them via real namespaces.
