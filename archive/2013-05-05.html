News & announcements

Symfony 2.3.0 Beta 1 released
Symfony 2.3 is nearly here, with a whole shed load of features and changes, so if you use Symfony or rely on components that do (increasingly more frameworks and individual components do) you might want to get testing.

phpMyAdmin 4.0.0 released
v4 of phpMyAdmin has gone gold, with lots of bug fixes rolled in, one of the biggest differences in 4 is that it no longer uses the well known split iframe interface and instead uses a single HTML document with a JavaScript driven tree UI.

php|architect April issue out
The April edition of php|a is out, this month featuring mostly articles about security, from password hashing done right to securing file upload. the team at musketeers.me are also running a survey to access which format people prefer the magazine in as they've switched back to the classic vertical style as an experiment.

Articles

An Introduction to Ctype Functions ♥
David Shirey simply covers various uses of the ctype extension for validating characters in a string.

Using RabbitMQ in Unit Tests ♥
A very clever write up by Alvaro Videla about how they uses the RabbitMQ pre-built nightly (currently only available for OS X, but this will hopefully change) to untar and run an out of the box instance of RabbitMQ with which to run their unit tests against. Very clever alternative to mocking out your RabbitMQ provider classes, or if you really want to test that part against a real component.

Testing Smells - Try/catch ♥
Chris "grmpyprogrammer" Hartjes gives us this great piece on why using try/catch blocks in your unit tests is nearly always bad practice, especially with inbuilt functionality in PHPUnit for handling exceptions as part of a test.

Serializing Data Like a PHP Session ♥
A quick piece by Josh Adell on serialising arbitrary data to/from the format used by PHP for storing session data, as the builtin functions session_encode and session_decode will only work directly with the $_SESSION superglobal. Josh provides some userspace functions to get around this that work by temporarily swapping data in $_SESSION.

WordPress 10th Anniversary Blogging Project ♥
Dougal Campbell kicks off discussion about WordPress's 10 year anniversary (on the 27th May) by writing about his history during the project's timeline, and encouraging others to the same. Use the hashtag #wp10 on Twitter to get in on the action.

Working With Dates and Times in PHP ♥
A great write up by Ravi Gill on various techniques for working with datetimes in PHP using the DateTime, DatePeriod and DateInterval classes that ship with 5.2 and above.

Use RouteMatch in Zend Framework 2 For Easy Routing ♥
Matthew Setter wrote up how to use the ZF2 RouteMatch, Router and Request APIs to dynamically change the parameters and filters used by Zend Paginator instance, displaying customised results in a paginated list, with a re-render.

Crop and Resize Images with ImageMagick ♥
A nice little tutorial by Sandeep Panda on using the ImageMagick extension to resize user uploaded images, for example automatically resizing a profile image, or even taking a user selected crop size (he covers using one of the many jQuery plugins to allow the user to select a crop size from their selected image). I would also like to note that while this is a great technique for a low impact service, e.g. for profile pictures that don't get set very often, if you are planning on doing something more image heavy (e.g. a photos site that might hit the cropping or resize functionality quite a lot) you should invest in offloading the image resizing and cropping to an out-of-request task system. As Sandeep mentions you should also put some time into making sure the file upload portion of the app is secure and has sane limits.

Functionally Testing Your Application Using Mink ♥
A great article by Konstantin Kudryashov on using the Mink browser emulation/instrumentation library to create functional/integration tests, covering hooking up to Silex using silex-mink, using it with PHPUnit tests rather than Behat, and getting the most interface/API coverage out of your tests. The Selenium driver is covered, but there are plenty of drivers available for different engines.

Testing and Contributing with Composer Packages ♥
A typically well written and thorough guide by Phil Sturgeon on contributing to PHP packages that use Composer, from getting a package installed locally with dependencies, to testing your patch and going through with a pull request. We need lots more of this in the community to get the present ecosystem of Composer packages fleshed out and more mature

7php Interview With Jakub Vrana Co-Author Of The Official PHP Manual ♥
Khayrattee Wasseem is back with another great interview with a PHP community luminary, this week Jakub Vrana (♥) is up, known for his work on the PHP Manual, Adminer and NotORM, covering the usual early days with PHP, how he's involved in the PHP community, to PHP hints and tips (especially some favourites from his book), his involved with the PHP manual and more.

How to gracefully handle cache expiration ♥
A good write up by Aleksey Korzun on being smarter with your caching, focusing on using the Memcache extension with memcached for caching some model objects, and solving common high load issues such as the thundering herd problem with techniques like wrapping your objects with shorted cache time and delaying cache returns out of the request cycle to avoid response delays caused by execution.


Projects

arrch
ORM-style queries for arrays, which might be just what you need if you're working with large and/or complex data structures imported into an Array.

ClosestDate
A helper utility that provides APIs for finding dates close to a given date (or list or dates), handy for calendar functionality and user scheduling.

Duct
A stream based JSON library which allows you to parse and utilise JSON data incrementally, such as when streamed from a network connection.

graze/event
An event based pub/sub library, allows you to subscribe to events by simply implementing a specific method or set of methods.
