News & announcements

Symfony 2.2.0 Beta 1 released
Lots of new and shiny features in 2.2, it looks like beta 1 introduces a feature freeze, so time to get testing, see the schedule for newness and an idea when to expect a stable release.

APC 3.1.14 released
The defacto PHP opcode cache has been bumped to 3.1.14, still in a beta state, but this release adds 5.5 support so perfect for testing with the new 5.5 alphas. Check out the changelog for more.

 

Audiovisuals

A Look Back on PHP 5.5 Development in 2012
This is a video visualisation of PHP 5.5 activity in the past year by one of the core devs, shown by displaying branch nodes representing source control branches and their committers, with nodes branching off and merging back in again. Take a look at gource if you're interested in making these videos yourself.

Programming with Anthony - Dependency Injection
Another excellent video tutorial by Anthony Ferrara, this time covering Dependency Injection, and to a lesser degree Inversion of Control, in PHP in Anthony's charming hand-drawn notes style.

Articles

PHP needs a vision
There's been a bit of a storm in a teacup again on the PHP internals mailing list, sparked off by a discussion about introducing proper annotations to the language (not just docblock parsing), and frankly some people getting their nickers in a twist. Linking to the HN page for this as there's been some good comments on there (just ignore the generic PHP hate that always comes with a HN discussion).

2012: A Year in PHP
Ben Longden looks back at his predictions from last year, going over hits and misses, and what he reckons 2013 might have in store. Some good points, especially regards improvements in PHP and community outlooks this past year, and solid predictions for the coming year.

Избавляемся от дублирования сквозного кода в PHP: рефакторинг кода с АОП
A very nice write-up (in Russian) by Лисаченко Александрon on using the Aspect Orientated Programming pattern in PHP in order to de-duplicate code across a project. He covers various different techniques such as proxy objects, signalling, and rounds out the first introductory part of this series (hopefully) with using the Go! library to implement AOP in PHP.

SunshinePHP is a community event
The guys behind SunshinePHP reply to recent misrepresentation of the conference as a commercial event, in their own words it is by developers, for developers, and all profits from the conference will go 100% into the user group and PHP community. So there, no go sign up. ;)

Why is FastCGI /w Nginx so much faster than Apache /w mod_php?
Kevin Schroeder takes the oft heard assertion "PHP is faster on Nginx + FastCGI" by running the actual numbers and tracing Apache to see why so many people might be seeing worse performance in Apache+mod_php. The answer probably won't surprise you, but the fact Apache+mod_php actually outperforms Nginx with the right configuration might do, read on for some good advice.

7php Interviews With Ligaya Turmelle Co-Founder Of PHPwomen.org
Khayrattee Wasseem back with another great interview with a PHP luminary (wasn't it just yesterday the last one way out?), this time Ligaya Turmelle of phpwomen.org fame, discussing everything from her first steps into PHP to her huge contributions to and involvement with the PHP community on whole. My favourite part is Ligaya remeniscing about an embarrassing email send test back at her old company, because I've been that exact same person (as I'm sure many of you have) before and can now laugh at it looking back.

PHP Apps and Platform as a Service
Andy Podner with a good write up of deploying your first PHP app to AppFog, the multi-language/enviroment Platform as a Service (PaaS) provider, complete with handy step by step guide, and comparison with other PaaS/IaaS providers with PHP offerings.

We are all PHP Evangelists
Rafael Dohms reposted his php|a article from last year in which he recounts a lovely little true story about a time in the Brazilian PHP community when everyone banded together to get some harmful misinformation spread by a TV station redacted and corrected. Good stuff and some truly heartfelt and true words on community.

reddit: Your prediction for PHP in 2013?
Does what it says on the tin: a reddit discussion of peoples predictions for the next year, some funny, some earnest, some best just glossed over.

Projects

php-cliopts
Billed as a no-nonsence command line options parser, nice and simple API.

AsymetriCrypt
A promising asymetric public/private key encryption library, no docs as yet but some easy to understand examples.

CalendR
An OO PHP calendar management library, allows you define calendar functions and manage date/time based events, one to watch.

Bob
A tiny and messy build build tool for PHP much in the same vein as Pake or Phing but without a lot of hassle (no XML and a cleaner API than Pake), which is actually something I'd considered making myself, and the perfect companion for Composer.

itertools
Reminiscent of itertools in Python-land this is a well defined library of utility functions for dealing with iterables in PHP and making them as useful as the all powerful Array.

Phinx
A database migration management library, allows you to define RDBMS independent data/schema migrations. +1!

Phue
Ever wanted to control the hue and illumination level of the lighting in your house from PHP? No? Well tough, because now you can, if you have a Philips Hue lighting system and this library that is.

Geocoder
A geocoding abstraction library with support multiple geo-aware API providers, so your app can know where the user is when they post pictures of kittens with funny captions.

howdoi
Actually an app written in Python, but a damn handy utility for PHP devs. To show what it does I'll give you an example instead:
$ howdoi reverse an array php
$newArray = array_reverse($theArray, true);
