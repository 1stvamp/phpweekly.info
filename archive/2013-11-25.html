News & announcements

php.net - Our modern web theme goes live!
The PHP web team finally pulled out of beta the new modern theme for everyone's favourite site and documentation, and my word is it smooooth. It's also quite a lot lighter than the old dark purple and some people have reported issues, but there is a fix.

RFC: Constant Scalar Expressions gone to the vote
This RFC to allow scalar expressions (aka changeable or dynamic expressions) to be used in constants (e.g. using the const key, of via defines etc.) is at the voting stage, and so far it's looking good for their inclusion in 5.6. Anything which makes PHP more dynamic at runtime with little cost to performance or readability is just great.

php[architect] — November 2013
The latest php|a is out for digital download and in print, this month focussing on frameworks, in particular some rising stars such as the Laravel and Adianti frameworks.

PHP UK - schedule up and early bird tickets still available
The premier UK PHP conference is shaping up to be another great event this year, with a packed schedule of interesting talks from some well known and not-so-well-known faces. There are still some early bird tickets up for grabs, so get 'em before they're gone.

Phil Sturgeon is releasing a new book on building APIs ♥

PSR-4 Autoloading proposal back up for vote

PHP 5.4.22 and PHP 5.5.6 released

Articles

Working with PHPUnit and Selenium Webdriver ♥
A good primer by @davert on using PHPUnit with Facebook's php-webdriver library to perform functional testing against a live instance an application, using real actions inside a Selenium instrumented Firefox instance, this is one of those areas that it's good to have knowledge in, but be careful in how you use such tests, you probably want them as part of your mainline integration testing and pre-deploy testing, but they can be a little too fragile for run every time CI tests.

Beyond Object Oriented Programming ♥
Another excellent article from Anthony Ferrara in which he discusses going beyond OOP concepts for the purposes of classes and how they're wired together, covering what can be learned and applied from the SOLID paradigm to the concept of behaviours, and introducing Functional OOP, or the combination of functional programming with objects, avoiding state, and the use cases of Value Objects. Extremely interesting and a lot for everyone to take away.

Managing schema changes with MongoDB ♥
Derrick Rethans discusses a few techniques for managing schema migrations in Mongo. Effectively it's mostly the same way you would with any relational DB, except you don't worry about the explicit schema migration itself, just data migration (as existing documents still have to be altered to match a new layout). On show we have the "write a migration script and watch it ruuuuuuuuun" method, and the "support both schemas in your code" method (with Derrick favouring an explicit schema versioning system vs detecting the doc layout). Some good takeaways for any DB here, although as someone points out in the comments the goal should be to move toward a fully migrated system with the old schema code eventually removed.

Using Git Hooks to delegate routine tasks ♥
Rafael Dohms tells us all how to setup simple hook scripts in a git repo to ensure you have the correct dependencies or have re-run preprocessors when switching git branches. Not explicitly PHP (Rafael uses running composer as an example), and you can also do similar with hooks in other system such as mercurial, but definitely something you should think about doing.

PHP: Cannot access property started with '\0' ♥
A succinct post by Christian Weiske on this confusing error which may catch you out if you're doing anything with casting objects to arrays, or a library you depend upon does. Some of the examples are white on grey so you may need to select the text to actually see anything.

Building A Sensor Phalanx With PHP ♥
Thomas Weinert  keeps on pushing the envelope on exactly how much hardware hacking/integration you can do with PHP, yet again showcasing his powerful Carica Io library, reading data from the analog sensors on an Arduino, and displays the results in a slick chart in a web browser. There is example code to grok, and a video to watch.

What is next for Symfony2? ♥
Lukas Kahwe Smith considers future directions for the Symfony2 community now that Symfony2 itself is pretty much hitting the sweet spot it's aimed at (with the exception of bug hunting that all projects constantly need, and some core classes here and there in need of a little lovin'). He'd like to spread the resources at hand amongst some of the more popular bundles available, given the lack of redundancy/diversity amongst maintainers, but he's asking for people to give their opinions.

A Peek At Aura v2 -- Aura.Router ♥
Paul M. Jones talks about the new Aura.Router package, with a quick summary of what a router and dispatcher are, what Aura.Router isn't and isn't (clue it isn't a dispatcher, there's a Aura.Dispatcher package for that), and the API it provides for turning paths, or even a series of server or request params, into a set of values that can then be used by a dispatcher to hand the request off to a controller.

Authentication With Laravel 4 ♥
A simple step by step guide by Andrew Perkins to creating a registration + login system in Laravel 4, complete with all the fiddly pre-amble steps like making sure Twitter bootstrap is setup and wiring up views.

Processing data with PHP using STDIN and Piping ♥
A fantastic write up by Greg Freeman on using STDIN, Unix pipes and PHP streams together to do simple multiple process communication and data streaming between different scripts (or standard CLI utils like cat, zcat, grep, zgrep etc.) Greg also covers a few edge cases with detecting a timed out pipe, and includes some nice workaround code.

Controlling a receipt printer with ZeroMQ and PHP ♥
Jachim Coudenys wrote up his recent experience developing a P.O.S. webapp that needed to communicate with a receipt printer connected to the checkout system. He came up with a very elegant solution using a set of processes listening/publishing on ZeroMQ sockets, and connected back to the server with the webapp on. Simple and avoids the need for scheduled tasks.

Security issues with using PHP's escapeshellarg ♥
You may think you're completely protected from exploits by wrapping any user data passed to a command with escapeshellarg however Eldar Marcussen shows us that isn't always the case if the command you're calling doesn't do any validation of it's parameters, in the example he provides he calls a PHP script masquerading as a PNG file by telling tar to use PHP as it's compression program (effectively just piping the file from tar to PHP), with the file containing a malicious bit of code that opens up a shell on a remotely connectable port. There's very little to be done to fix this from PHP, but it's something everyone should be aware of as a possible issue.

Juggle Chainsaws, Not Types ♥
Samantha Quiñones summarises, extremely well, just how insane the weak typing in PHP from the infamous comparison chart to the parade of vulnerabilities in popular open source project that have started out as someone making quite well reasoned assumptions about how PHP would compare one variable's type to another. She also recites a recent tale of woe involving in_array and how it may catch you out in a familiar way.

Using PHP Streams Effectively ♥
A very extensive walkthrough by Vito Tardia on using stream filters to do clever things to files and other input/output streams, with an excellent example of using a stream filter to parse/format Markdown text automatically.


Projects

phpdbg ♥
An in-their-own-words awesomesauce PHP debugger built for PHP 5.4+, with a tonne of powerful features, including a userland PHP API for interacting/setting up the debugger from inside your apps. There is talk of including this in php-core in the future, so keep an eye on this one.

pho ♥
A young project but with lots of promise, pho is Jasmine/RSpec inspired BDD framework with a very elegant API.

Plates ♥
A really well made non-compiled templating framework (e.g. PHP is the templating "language"), reminiscent of Savant, with lots of really nice features. This project (like Monga below) is also part of the The League of Extraordinary Packages so you know it's up to scratch quality wise.

Patchwork UTF-8 for PHP ♥
A very extensive wrapper around various UTF-8/charset libraries that may or may not be present on your target host, with provided userland versions of all but one (PCRE, which is the most commonly available), with the added bonus of grapheme cluster awareness and handling.

Monga ♥
Although in my day job I probably see more new MongoDB abstractions, ODMs and the like each week than I could possibly even use in a lifetime, Monga stands out as being a very thin yet useful wrapper around the native MongoDB client extension. Of real note is the clever use of clojures in query calls to provide a persistent and fluent context object.
